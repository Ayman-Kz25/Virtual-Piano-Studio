# 🎹 Virtual Piano Studio

A beautiful and interactive virtual piano built with HTML, CSS, JavaScript, and Tone.js. Play notes using your keyboard or mouse, switch between different instruments, adjust the volume, and enjoy a soft animated background that reacts to your music.

> Live Demo: **https://virtual-piano-studio-blush.vercel.app/**

## Features

- 🎵 Play piano notes using your keyboard
- 🖱️ Click piano keys with your mouse
- 🎹 Supports white and black keys
- 🎼 Multiple instruments
  - Piano
  - Electric Piano
  - Organ
- 🔊 Real-time volume control
- 💡 Power On/Off functionality
- ✨ Animated aesthetic background
- 🫧 Floating bubbles and glowing blobs
- 💫 Ripple effects when notes are played
- 📱 Responsive layout for smaller screens

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Ayman-Kz25/Virtual-Piano-Studio.git
```

### Navigate to the project

```bash
cd Virtual-Piano-Studio
```

### Run the project

Simply open `index.html` in your browser.

Or use VS Code's Live Server extension for the best experience.
